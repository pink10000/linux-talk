# linux-talk
slides for intro to linux talk for ucsd tqt

# how to run

## dependencies
- [presenterm](https://github.com/mfontanini/presenterm), 
- mermaid-cli
- a kitty image protocol compliant terminal (like kitty or ghostty)

then run with 
```
presenterm slides.md -x --present
```
